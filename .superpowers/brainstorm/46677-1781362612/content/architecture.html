整体架构设计

Next.js App Router + Zustand + Tailwind CSS

页面路由（App Router）

Route Structure
/ 开始界面 StartPage
/setup 比赛设置 SetupPage
/game 比赛进行中 GamePage
/result 排名结果 ResultPage

组件树

Component Tree
src/
├── app/
│   ├── layout.tsx          # 根布局 (字体/全局样式)
│   ├── page.tsx            # 开始界面
│   ├── setup/page.tsx      # 设置界面
│   ├── game/page.tsx       # 比赛界面
│   ├── result/page.tsx     # 结果界面
│   └── api/
│       └── words/
│           ├── route.ts    # POST 解析 Excel
│           └── route.ts    # GET  获取单词列表 (脚手架)
│
├── components/
│   ├── ui/                 # 通用 UI 组件
│   │   ├── Button.tsx
│   │   ├── Modal.tsx
│   │   └── Countdown.tsx
│   ├── game/               # 比赛相关组件
│   │   ├── GameLayout.tsx        # 多人布局容器
│   │   ├── PlayerArea.tsx        # 单个选手答题区
│   │   ├── WordBlock.tsx         # 单词卡片块
│   │   ├── BlockGrid.tsx         # 块网格排列
│   │   ├── PlayerTimer.tsx       # 选手计时器
│   │   └── RefereeToolbar.tsx    # 裁判工具栏
│   ├── setup/              # 设置相关组件
│   │   ├── PlayerCountSelector.tsx
│   │   ├── WordCountInput.tsx
│   │   └── ExcelUploader.tsx
│   └── result/             # 结果相关组件
│       ├── RankCard.tsx
│       └── Podium.tsx
│
├── stores/
│   ├── gameStore.ts        # 比赛核心状态 (Zustand)
│   └── setupStore.ts       # 设置状态 (Zustand)
│
├── lib/
│   ├── wordParser.ts       # Excel 解析逻辑
│   ├── gameLogic.ts        # 抽词/打乱/匹配判断
│   └── storage.ts          # localStorage/IndexedDB 工具
│
└── types/
    └── index.ts            # WordPair, Block, Player 等类型
